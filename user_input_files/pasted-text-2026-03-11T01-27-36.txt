Aquí tienes los comandos para entrar al bench console dentro de tu container Docker:

1. Primero, identifica el container del backend:

bash
docker ps | grep backend
2. Entra al container:

bash
docker exec -it <nombre_container_backend> bash
(Generalmente algo como erp-sysmayal2-backend-1 o similar)

3. Una vez dentro, abre bench console:

bash
cd /home/frappe/frappe-bench
bench --site erp.sysmayal2.cloud console
4. Ya dentro de la consola Python, puedes correr scripts para detectar/mergear duplicados:

python
# Ver clientes duplicados por customer_name
from collections import Counter
customers = frappe.get_all("Customer", fields=["name", "customer_name"], limit_page_length=0)
names = [c.customer_name.strip().upper() for c in customers]
dupes = {k: v for k, v in Counter(names).items() if v > 1}
for name, count in sorted(dupes.items(), key=lambda x: -x[1]):
    print(f"{name}: {count} veces")
python
# Merge de un cliente duplicado (ejemplo)
frappe.rename_doc("Customer", "BARENTZ Italia S.p.A. - 1", "BARENTZ Italia S.p.A.", merge=True)
frappe.db.commit()
python
# Script para listar todos los duplicados con sus IDs
for name, count in sorted(dupes.items(), key=lambda x: -x[1]):
    matches = [c.name for c in customers if c.customer_name.strip().upper() == name]
    print(f"\n--- {name} ({count}) ---")
    for m in matches:
        print(f"  {m}")
