1. What This Is
A multi-agent system running inside Raven Chat (ERPNext's built-in messaging) that responds to @ai commands. It manages a full Sales-to-Payment pipeline: Quotation → SO → Manufacturing WO → Stock Entry → Delivery Note → Sales Invoice → Payment Entry.

Entry point: raven_ai_agent.api.agent.handle_raven_message (registered in hooks.py)

2. Architecture Overview

3. RaymondLucyAgent — Intelligence Engine
The core agent uses 4 named protocols:

Protocol	Name	What It Does
Raymond	Anti-Hallucination	NEVER fabricate ERPNext data — always query DB first, cite doc names, express confidence (HIGH/MED/LOW)
Memento	Persistent Memory	tattoo_fact() stores facts to AI Memory doctype with optional vector embeddings
Lucy	Context Continuity	get_morning_briefing() loads memories + last session summary at start; end_session() saves summary
Karpathy	Autonomy Slider	L1=read-only, L2=modify-with-confirm, L3=dangerous-ops. ! prefix = force mode
LLM: OpenAI GPT-4o-mini, temperature 0.3
Settings cascade: AI Agent Settings doctype → Raven Settings → frappe.conf
Custom Doctypes: AI Memory (persistent facts/summaries), AI Agent Settings (config)

7. Hard-Won Rules (Read Before Coding)
#	Rule	Why
1	No import frappe in Server Scripts	ERPNext sandbox pre-injects frappe; explicit import → ImportError
2	bench restart ≠ Python reload	Must use docker restart for Python changes to take effect
3	Git remote is upstream	git pull upstream main — origin doesn't exist or points elsewhere
4	Regex breaks in JSON payloads	Double-escaping kills \d → \\d. Use keyword matching for intent detection
5	ERPNext v16 make_stock_entry()	Returns frappe._dict, not Document — wrap with frappe.get_doc("Stock Entry", result.get("name"))
6	SO names have spaces	SO-00753-GREENTECH SA — regex must use negative lookahead: r'(SO-[\w\-]+(?:\s+(?!from\b|to\b|...\b)[\w\.]+)*|SAL-ORD-\d+-\d+)'
7	Fix agent.py, not base.py	base.py is dead code — agent.py execute_workflow_command() is what runs
8	Quotation = truth source	SO must be validated/synced against quotation, especially payment terms
9	All bots use @ai prefix	Single entry point; internal routing by keyword matching
10	! prefix = force mode	Skips confirmation dialogs (like sudo)
