# Release v9.1.0 - Migration Safety Release

## Overview
This release fixes critical migration safety issues where custom doctypes were being incorrectly deleted during `bench migrate` operations. After this fix, all custom doctypes are properly preserved.

## Breaking Changes
None. This is a bug fix release.

## Migration Fixes

### Controller Class Name Corrections

### File Structure Repairs
- Created missing `.js` files for all doctypes
- Ensured proper `__init__.py` files exist in doctype directories
- Removed broken patch references from `patches.txt`
- Cleaned up invalid `override_doctype_class` from `hooks.py`
