# v9.1.0 Release Notes - BOM Hierarchy & AI Agent Release

**Release Date:** 2026-02-17  
**Branch:** `feature/v9.1.0-development`

---

## 🎯 Overview

Version 9.1.0 introduces comprehensive BOM (Bill of Materials) hierarchy management with AI-powered tracking through the Raven agent system. This release also includes repository cleanup and documentation updates.

---

## ✨ New Features

### BOM Tracking AI Agent
- **New Raven Commands:**
  - `bom health` - Run comprehensive BOM health check
  - `bom inspect <BOM>` - Inspect specific BOM structure and hierarchy
  - `bom status <ITEM>` - Get BOM status for an item
  - `bom issues` - List all known BOM issues from tracking database

### BOM Health Monitoring
- Automated health checks via `bom_status_manager.py`
- Known issues tracking via `bom_known_issues.json`
- Weekly scheduled health reports via `scheduled_bom_health.py`
