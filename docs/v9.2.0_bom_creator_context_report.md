## 4. Critical Business Rules

### 4.1 Container Handling (IMPORTANT)
```
┌─────────────────────────────────────────────────────────────────┐
│  CONTAINERS ARE ITEMS, NOT UOMs                                  │
├─────────────────────────────────────────────────────────────────┤
│  ❌ WRONG: UOM = "IBC" or UOM = "Drum"                          │
│  ✅ RIGHT: Item = "E001" (220L Barrel) with UOM = "Piece"       │
│  ✅ RIGHT: Item = "E011" (IBC Container) with UOM = "Piece"     │
└─────────────────────────────────────────────────────────────────┘
```

**Container Items:**
| Item Code | Description | Capacity |
|-----------|-------------|----------|
| E001 | 220L Barrel Blue | 220L (~238 Kg) |
| E011 | IBC Container | 1000L (~1080 Kg) |
| E012 | Reused IBC Container | 1000L (~1080 Kg) |

### 4.2 Fractional Quantities
The BOM uses fractional container quantities to represent usage per Kg of output:

```
Container Qty = 1 / Container Capacity (Kg)

Example (E001 - 220L Barrel):
- Capacity: ~230 Kg
- Qty per 1 Kg output: 0.004351 Piece
- For 2000 Kg production: 2000 × 0.004351 = 8.7 Barrels ≈ 9 Barrels
```

### 4.3 Equivalent Powder Measurement
Liquid products (0227) are measured by their **equivalent powder weight**, not actual liquid weight:

```
Example:
- 2000 Kg equivalent powder (0303)
- = 18,000 Kg concentrated juice (0227)
- = 36,000 Kg fresh aloe leaf (M033)
```
